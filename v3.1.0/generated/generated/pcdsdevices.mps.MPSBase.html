pcdsdevices.mps.MPSBase¶

class pcdsdevices.mps.MPSBase(*args, veto=False, **kwargs)¶

Base MPS class.

Used for shared methods between the individual MPS bit class and the MPSLimits class. The class handles much of the bookkeeping for both of these classes.

Each subclass must reimplement:

  • faulted()

  • bypassed()

  • _sub_to_children()

Attributes

SUB_FAULT_CH

tripped

Whether this will trip the MPS system.
