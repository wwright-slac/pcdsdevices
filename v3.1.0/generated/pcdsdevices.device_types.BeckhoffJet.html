pcdsdevices.device_types.BeckhoffJet¶

class pcdsdevices.device_types.BeckhoffJet(*args, **kwargs)¶

Full liquid jet setup controlled by a Beckhoff PLC.

This includes three axes for the jet manipulator, two axes for each of two slits, and a single axis for the detector. Each of their PVs will be inferred from the base prefix.

Parameters

  • prefix (str, optional) – Liquid jet base PV.

  • name (str) – A name to refer to the device.
