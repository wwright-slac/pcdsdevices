pcdsdevices.state.CombinedStateRecordPositioner¶

class pcdsdevices.state.CombinedStateRecordPositioner(prefix, *, name, **kwargs)¶

A StatePositioner for an X/Y combined state positioner EPICS record.

Includes x_motor and y_motor attributes for motor level access on two-axis positioners.

states_list does not have to be provided.
